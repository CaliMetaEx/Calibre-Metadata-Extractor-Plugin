# Contributing

Vielen Dank für dein Interesse, zum Calibre-Metadata-Extractor-Plugin beizutragen! Bitte beachte die folgenden Hinweise:

1. Lizenz: Mit dem Einreichen eines Pull Requests erklärst du, dass dein Beitrag unter derselben Lizenz (`GPL-3.0-or-later`) stehen darf wie dieses Projekt. Falls du nicht der alleinige Urheber deines Beitrags bist, stelle sicher, dass alle Mitautoren zustimmen.

2. Datei-Header: Bitte füge neue oder geänderte Quelltextdateien mit folgendem Header ein (Beispiel für Python):

```python
# SPDX-License-Identifier: GPL-3.0-or-later
# Copyright (C) <year> <author name or handle>
```

3. Tests und Stil: Wenn möglich, füge Tests oder eine kurze Anleitung hinzu, wie man deine Änderung lokal testet. Halte dich an bestehende Code‑Stil‑Konventionen.

4. Commit‑Nachrichten: Schreibe aussagekräftige Commit‑Nachrichten und fasse zusammen, was geändert wurde und warum.

5. Sign-offs (optional): Wenn du willst, kannst du einen DCO-Sign-off verwenden (addiere `Signed-off-by: Name <email>` in die Commit‑Beschreibung). Das ist optional, aber hilft bei der Nachverfolgbarkeit.

Wenn du Fragen zu Rechten oder Lizenz hast, schreibe ein Issue und beschreibe kurz dein Anliegen.
